# Computing Technology — Developing Apps and Web Software

Code samples and walkthroughs for the **Developing Apps and Web Software** unit (Year 10 Computing Technology, NSW Stage 5).

> Lesson plans, assessments, and announcements live in **Canvas**. This repo is just the code.

---

## Lessons

| # | Topic | Folder |
|---|-------|--------|
| 1 | HTML basics — building a page from scratch | [`lesson-1/`](lesson-1/) |
| 2 | CSS basics — styling the same HTML two ways | [`lesson-2/`](lesson-2/) |

Each lesson folder contains the demo files used in class and any starter files you'll need to follow along.

### Lesson 1 — HTML basics
- [`demo/skeleton.html`](lesson-1/demo/skeleton.html) — the bare HTML document skeleton
- [`demo/scaffold.html`](lesson-1/demo/scaffold.html) — guided TODOs for your first page
- [`demo/example-complete.html`](lesson-1/demo/example-complete.html) — a finished example page

### Lesson 2 — CSS basics
- [`starter/index.html`](lesson-2/starter/index.html) + [`starter/starter.css`](lesson-2/starter/starter.css) — your starting point
- [`demo/styled_website.html`](lesson-2/demo/styled_website.html) + [`demo/styles.css`](lesson-2/demo/styles.css) — the live-demo page from class

## Reference

The [`reference/`](reference/) folder is your go-to whenever you're stuck. Browse any time, not just during the lesson it was introduced.

| File | Use it when... |

## How to use this repo

- **Read on github.com** — markdown files render as web pages, click straight through.
- **Download to run code** — green `< > Code` button → *Download ZIP*. Open the folder in VS Code and use Live Server to view HTML files in a browser.
- **Fork** if you want your own copy to modify and publish via GitHub Pages.
