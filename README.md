# Computing Technology — Developing Apps and Web Software

Code samples, snippet patterns, and student walkthroughs for the **Developing Apps and Web Software** unit (Year 10 Computing Technology, NSW Stage 5).

> **Lesson plans, assessments, and announcements live in Canvas.**
> This repo contains the **code** — sample HTML/CSS, reference patterns, and lesson walkthroughs you can read, clone, or fork. If you're ever working with code in this unit, expect to find it here.

---

## How this repo is organised

| Folder | What's in it |
|--------|--------------|
| `02-before-after-css/` | The "same HTML, two stylesheets" hook from Lesson 2. Open `before.html` and `after.html` side by side. |
| `03-snippet-patterns/` | Reference snippets for typography, colour, and a CSS glossary. Browse these any time you're stuck on a styling decision. |
| `lesson-2/` | Lesson 2 catch-up walkthrough, starter file for support tier, and the live-demo HTML. |

(More folders will be added as the unit progresses.)

## Viewing the styled examples (GitHub Pages)

Markdown files render directly on github.com — just click them.

HTML files (the actual styled demo pages) need GitHub Pages enabled to view them rendered in a browser. **Once Pages is enabled** (`Settings → Pages → Source: main, root`), the URLs follow this pattern:

```
https://[org-or-user].github.io/[repo-name]/02-before-after-css/before.html
https://[org-or-user].github.io/[repo-name]/02-before-after-css/after.html
https://[org-or-user].github.io/[repo-name]/lesson-2/live-demo/
```

If you want students to publish their own work, they can fork this repo, push their changes, and enable Pages on their fork. They'll get their own URL.

## How students should use this repo

1. **Browse on github.com** for reading — walkthroughs and references render nicely as web pages.
2. **Clone or download** when you want to actually run code. Open the folder in VS Code and use Live Server.
3. **Fork** if you want your own copy to modify and publish via GitHub Pages.

For students new to GitHub: the green **`< > Code`** button at the top of any folder gives you a "Download ZIP" option. No account required.
